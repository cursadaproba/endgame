8.1. Uso de Excel/Google Sheets para Cálculos Matemáticos Básicos

8.1.1. Conceptos Fundamentales

Las hojas de cálculo te permiten organizar datos en celdas (intersección de filas y columnas) y automatizar cálculos mediante fórmulas y funciones.

  • Celdas: Cada celda se identifica con una letra (columna) y un número (fila), por ejemplo, A1, B3.
  • Fórmulas: Se escriben iniciando con “=”. Ejemplo: =A1+B1.
  • Funciones: Bloques predefinidos para realizar tareas comunes, como =SUMA(A1:A5) o =PROMEDIO(A1:A5).
  • Referencias: Pueden ser relativas (ajustables) o absolutas (fijas, usando “$”, por ejemplo, $A$1).

8.1.2. Operaciones y Funciones Comunes

Entre las operaciones básicas se incluyen:

  • Suma: Usá “+” o =SUMA(A1:A5).
  • Resta: Usá “-”.
  • Multiplicación: Usá “*”.
  • División: Usá “/”.
  • Potenciación: Se indica con “^”. Ejemplo: =A1^2.
  • Raíz cuadrada: Usá =RAIZ(A1).
  • Porcentaje: El símbolo “%” se usa para cálculos de porcentajes, por ejemplo, =A1*10%.

8.1.3. Ejemplo Práctico: Trayectoria de un Proyectil

Podés usar Excel para calcular la trayectoria de un proyectil. Por ejemplo:

  • Columna A: Tiempo (t) en segundos (0, 1, 2, …).
  • Columna B: Velocidad horizontal \( V_x \) (constante) y vertical \( V_y \) (calculada a partir de la velocidad inicial y el ángulo).
  • Columna C: Posición horizontal \( x = V_x \times t \).
  • Columna D: Posición vertical \( y = V_y \times t - \frac{1}{2}gt^2 \).

Usá las funciones matemáticas de Excel para automatizar estos cálculos y, finalmente, crea un gráfico de dispersión para visualizar la trayectoria.

8.2. Introducción a Python para Simulaciones de Probabilidad

8.2.1. Conceptos Básicos

Python es un lenguaje de programación que te permite simular experimentos y analizar datos. Entre sus ventajas destacan la facilidad de manejo de variables, estructuras de control y el acceso a poderosas bibliotecas como NumPy, SciPy y Matplotlib.

8.2.2. Ejemplos de Simulación

Por ejemplo, podés simular el lanzamiento de un dado usando:
random.randint(1,6)
y graficar los resultados con Matplotlib para comparar la frecuencia experimental con la teórica.

8.3. Uso de Motores de Videojuegos (Unity/Unreal) para Eventos y Simulaciones Físicas

8.3.1. Conceptos Básicos

Los motores de videojuegos integran herramientas para gráficos, físicas y scripting. Unity utiliza C# y Unreal Engine utiliza C++ y Blueprints. Estos motores permiten implementar eventos aleatorios y simulaciones en tiempo real.

8.3.2. Ejemplos Prácticos

Por ejemplo, en Unity podés usar Random.Range() para generar posiciones aleatorias de enemigos o simular trayectorias de proyectiles mediante componentes como Rigidbody y Collider.

8.4. Ejercicios Aplicados a Videojuegos

A continuación se presentan ejercicios prácticos que te ayudarán a familiarizarte con el uso de las herramientas computacionales, especialmente Excel, para aplicar conceptos matemáticos en videojuegos.

Ejercicio 8.1: Simulación de Tiradas de Dados en Excel

Enunciado:

Usando Excel o Google Sheets, crea una hoja que simule el lanzamiento de dos dados de 6 caras. Utiliza la función =ENTERO(ALEATORIO()*6)+1 para generar un número entre 1 y 6. Realizá 1000 tiradas y calcula la frecuencia de cada suma (de 2 a 12). Finalmente, crea un gráfico de columnas que compare la frecuencia experimental con la teórica.

Objetivo: Familiarizarte con funciones aleatorias, fórmulas y gráficos en hojas de cálculo.

Ejercicio 8.2: Cálculo de Probabilidades en Excel

Enunciado:

Utiliza funciones de combinatoria en Excel para calcular la probabilidad de obtener una Escalera Real en póquer. Usa fórmulas como =COMBIN(52,5) para el total de manos y calcula el número de manos que forman Escalera Real. Luego, determina la probabilidad dividiendo el número de manos favorables entre el total.

Objetivo: Aprender a usar funciones combinatorias y a calcular probabilidades paso a paso en Excel.

Ejercicio 8.3: Simulación de un Sistema de "Loot Boxes" en Excel

Enunciado:

Diseñá una hoja de cálculo que simule la apertura de "loot boxes" en un juego. Define las probabilidades para diferentes rarezas (por ejemplo, 60% común, 25% raro, 10% épico y 5% legendario) y utiliza la función =ALEATORIO() para asignar un resultado a cada apertura. Realizá 1000 aperturas y utiliza tablas y gráficos para comparar la frecuencia experimental con la teórica.

Objetivo: Aplicar funciones aleatorias y de probabilidad en Excel para simular eventos y analizar resultados.

Ejercicio 8.4: Análisis de Datos y Gráficos

Enunciado:

Importá datos de un juego (por ejemplo, estadísticas de daño o experiencia) en Excel. Utilizá funciones estadísticas como =PROMEDIO(), =MEDIANA() y =DESVEST() para analizarlos. Luego, crea gráficos dinámicos (gráficos de barras y líneas) para visualizar la evolución de estas estadísticas.

Objetivo: Aprender a analizar datos y a crear dashboards interactivos en Excel.

8.5. Resolución Paso a Paso

A continuación se muestra la resolución paso a paso de un ejercicio en Excel para ayudarte a entender el proceso.

Ejemplo Resolutivo: Tiradas de Dados

Problema: Simular 1000 tiradas de dos dados y graficar la frecuencia de las sumas.

  1. Generar Números Aleatorios: En la celda A2, escribe la fórmula =ENTERO(ALEATORIO()*6)+1 para simular un dado.
  2. Copiar la Fórmula: Copia la fórmula en las celdas A2:A1001 para obtener 1000 resultados. Haz lo mismo en la columna B para el segundo dado.
  3. Calcular la Suma: En la celda C2, escribe =A2+B2 y copia esta fórmula hasta la celda C1001.
  4. Contar Frecuencias: Utiliza la función =CONTAR.SI(C$2:C$1001, valor) para cada suma posible (2 a 12) en celdas separadas.
  5. Crear el Gráfico: Selecciona los datos de frecuencia y crea un gráfico de columnas para visualizar la distribución. Compara la frecuencia obtenida con la probabilidad teórica (por ejemplo, la suma 7 tiene mayor probabilidad).

8.6. Conclusión del Módulo 8

En este módulo aprendiste a utilizar herramientas computacionales para automatizar cálculos, realizar simulaciones y visualizar datos de forma interactiva. En particular:

  • Excel/Google Sheets: Son herramientas poderosas para organizar datos, realizar cálculos mediante fórmulas y funciones, y crear gráficos dinámicos para el análisis de información en videojuegos.
  • Python: Te permite simular experimentos y analizar datos de forma programática, complementando el trabajo realizado en hojas de cálculo.
  • Motores de Videojuegos (Unity/Unreal): Integran la física y generan eventos aleatorios para simular mecánicas de juego en tiempo real.

Estas herramientas, junto con los conceptos matemáticos que ya conocés, te brindan un conjunto robusto de habilidades para transformar la teoría en simulaciones y mecánicas de juego realistas.
